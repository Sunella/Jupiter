Wordpress video sharing site A website for all corpnet users to create channels, subscriptions, and personal playlists/favorites (can be public or private). Should be able to upload, download, share (specific video, or playlist), search, comment and rate a video. Will be focussed on the front end, i.e., to provide an interface to play videos already on openifs as well as to upload videos back to openifs.
